## 📈 Usage Metrics

### Installation
```bash
# Quick install (3 steps)
curl -sSL https://raw.githubusercontent.com/itsatif/hyperbrain-skills/main/install.sh | bash

# Manual install
git clone https://github.com/itsatif/hyperbrain-skills.git ~/.claude/skills/hyperbrain-skills
cd ~/.claude/skills/hyperbrain-skills
./install.sh
```

### Skill Invocation
Skills are invoked automatically by Claude Code based on:
1. **User request type** (e.g., "Create PRD" → Google L7 PM persona)
2. **Technical context** (e.g., "Angular component" → Angular patterns)
3. **Mentioned technologies** (e.g., "MQTT" → IoT patterns)
4. **Trigger keywords** (e.g., "Use Morpheus" → Morpheus MCP)

### Learning System
```bash
# Initialize memory
node ~/.claude/skills/hyperbrain-skills/self-learning/memory.js init

# View statistics
node ~/.claude/skills/hyperbrain-skills/self-learning/memory.js stats

# Export memory
node ~/.claude/skills/hyperbrain-skills/self-learning/memory.js export

# Optimize patterns
node ~/.claude/skills/hyperbrain-skills/self-learning/memory.js optimize
```

## 🚀 Performance Characteristics

### Activation Flow
```
User Request
    ↓
Superpowers Brainstorming (always activates first)
    ↓
Expert Persona Selection (based on task type)
    ↓
Technical Skill Application (based on context)
    ↓
Knowledge Base Query (via MCP if needed)
    ↓
On-Demand MCP Activation (if triggered)
    ↓
Self-Learning Capture (stores context)
    ↓
Response Generation (personalized)
```

- **Complex queries:** 5-10 seconds (MCP + knowledge base)
